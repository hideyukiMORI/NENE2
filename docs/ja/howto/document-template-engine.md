# ハウツー: ドキュメントテンプレートエンジン

`{{変数}}` 置換と管理者ゲートの書き込みを持つテンプレート CRUD を実演します。
フィールドトライアル: FT197（`../NENE2-FT/templatelog/`）。

## パターンまとめ
- テンプレートに `UNIQUE(name)` 制約 → 重複時に 409
- 一覧エンドポイントはペイロードを削減するために `body` を除外する
- `POST /templates/{id}/render` は `vars` オブジェクトを受け取り、`{{key}}` プレースホルダーを置換する
- 未知の変数はそのまま残す（エラーなし）
- 管理キーが作成/更新/削除をゲートする。レンダリングはパブリック
